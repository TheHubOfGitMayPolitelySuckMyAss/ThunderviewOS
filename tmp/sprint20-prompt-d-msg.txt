Prompts A-C, all 636 boxes had populated columns but empty Contacts —
zero recipients. Prompt D plumbs Contacts in.

- Bootstrap: teamKey resolved from pipeline.teamKey (no /users/me/teams
  endpoint exists despite docs implying otherwise)
- Push primitives: after box+fields, attach a Contact (member: best
  non-bounced email or primary; application: app.email)

- Forward path verified end-to-end on a synthetic application

Notable Streak API gotchas captured in code comments + CLAUDE.md:
- updateContact path is /v2/contacts/{key}, NOT /v2/teams/{}/contacts/{}
- addContactToBox body is { contacts: [{ key }] } — array of objects with
  the literal field name "key". String arrays and other shapes silently
  fail with 200 + null response
